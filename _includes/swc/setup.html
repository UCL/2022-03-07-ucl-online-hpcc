SSH

SSH is a tool that allows us to connect to and use a remote computer as our own. Please follow the directions below to install an SSH client for your system. Windows: Install MobaXterm from http://mobaxterm.mobatek.net. Get the Home edition (Installer edition). macOS: Although macOS comes with SSH pre-installed, you will need to install XQuartz to enable graphical support. Note that you must restart your computer to complete the installation.
